# 🧩 Aliucord Plugins

[![Plugins](https://img.shields.io/badge/Plugins-1-blue?style=for-the-badge)](https://github.com/arafatislam/aliucord-plugins)

## 🚀 Setup

> [!IMPORTANT]
> **Manual Installation Instructions**
> 1. Download the `.zip` file from the table below.
> 2. Move the file to: `Aliucord/plugins` folder on your device.
> 3. Restart Aliucord.

## 🧩 Available Plugins

| Plugin | Version | Description |
|:-------|:--------|:------------|
| [**AnotherButton**](https://raw.githubusercontent.com/arafatislam/archive/main/plugin/AnotherButton.zip) | `2.0.1` | A customizable chat button with folders, macros, and automation tools. |
